# Project 2 – Dynamic Report

## Overview

This project was completed for **STATS 220** at the University of Auckland.

The aim of this project was to create a dynamic report using **R Markdown**. The report combines text, code, data visualisation and automatically generated results in a reproducible workflow.

## Skills

- R
- R Markdown
- tidyverse
- Data visualisation
- Dynamic reporting

## Files

- `exploration.R` – Data exploration and analysis
- `dynamic_report.Rmd` – Source code for the dynamic report
- `dynamic_report.html` – Generated dynamic report
- `project2_report.Rmd` – Final project report
- `project2_report.html` – Final project report in HTML


## Project Summary

This project analyses travel patterns observed outside Scott Point School.

The dataset records:

- Time period (Morning, Midday, Afternoon, Evening)
- Transport mode (Walking, Car, Bicycle, Scooter, Bus)
- Travelling group (Alone, Pair, Group)
- Direction of travel (To school / From school)

The analysis was completed using tidyverse and presented as a reproducible HTML report generated with R Markdown.

## Learning Outcomes

- Developed reproducible reports using R Markdown
- Performed exploratory data analysis with tidyverse
- Created data visualisations to communicate findings
- Integrated code, figures and narrative into a single report
