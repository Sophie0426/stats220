# Project 5 – Web Scraping

## Overview

This project was completed for **STATS 220** at the University of Auckland.

The project collected climate-related media releases from the New Zealand Government Beehive website using web scraping techniques. The scraped data were cleaned, combined with information from Wikipedia, and analysed to produce visualisations of minister involvement in climate-related announcements.

## Skills

- Web scraping (rvest)
- tidyverse
- Data cleaning
