# Project 3 – Photo Analysis

## Overview

This project was completed for **STATS 220** at the University of Auckland.

The project explores a collection of selected photographs using R. Data were processed and visualised to identify interesting patterns, and the results were presented in a reproducible HTML report.

## Skills

- R
- tidyverse
- Data visualisation
- Data exploration
- R Markdown

## Files

- `exploration.R` – Data exploration and analysis
- `selected_photos.csv` – Dataset
- `project3_report.Rmd` – Source report
- `project3_report.html` – Final HTML report
- `top_photos.png` – Visualisation
- `creativity.gif` – Creative component

## Learning Outcomes

- Performed exploratory data analysis
- Created visualisations using R
- Produced reproducible reports with R Markdown
