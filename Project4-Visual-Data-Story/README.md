# Project 4 – Visual Data Story

## Overview

This project was completed for **STATS 220** at the University of Auckland.

The project presents a visual data story based on observations collected outside Scott Point School. Interactive visualisations and animations were used to communicate travel patterns across different times of the day.

## Skills

- R
- tidyverse
- ggplot2
- R Markdown
- Data storytelling
- Data visualisation

## Files

- `data_visualisation.R` – Data processing and visualisation
- `visual_data_story.Rmd` – Source code for the visual data story
- `visual_data_story.html` – Final interactive HTML page
- `project4_report.Rmd` – Project report
- `project4_report.html` – Final report
- `transport_animation.gif` – Animated visualisation

## Key Findings

The visual data story explores:

- Morning travel patterns
- Midday activity
- Afternoon observations
- Evening travel behaviour

The project demonstrates how data visualisation can communicate insights through an engaging narrative.
